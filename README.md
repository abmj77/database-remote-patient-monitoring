# 🗄️ Database Labs & Project – Remote Patient Monitoring System

## 🎯 Goal of This Work

This repository contains a collection of **SQL labs and a final database project** developed for a **Database course**.

The goal of these exercises is to build a strong understanding of relational database design and SQL operations by implementing practical examples and a real-world inspired system.

The project focuses on designing a **Remote Patient Monitoring System**, which allows doctors to monitor patients' health conditions remotely using recorded vital signs and consultation data.

---

## 📚 Learning Objectives

These labs and the project aim to practice and understand:

- Database schema design
- Table creation using SQL
- Primary and foreign key relationships
- Data insertion and modification
- SQL queries and filtering
- Aggregate functions and joins
- Subqueries
- Stored procedures in Oracle SQL
- Real-world database system design

All implementations were written and tested using **Oracle SQL**.

# 🏥 Final Project  
## Remote Patient Monitoring Database System

The final project implements a **database system for monitoring patient health remotely**.

The system records:

- Doctors
- Patients
- Consultations
- Vital signs monitoring
- Medical determinations
- Prescriptions
- Medicines

This allows healthcare providers to track patient conditions and manage consultations more efficiently.
